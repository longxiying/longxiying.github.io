TTQ (9F66)

TTQ byte 2 bits 8-7 are transient values, and reset to zero at the beginning of the transaction. All other TTQ bits are static values, and not modified based on transaction conditions.

Tag: 9F66
Length: 4
Value:
Byte 1
b8 b7 b6 b5 b4 b3 b2 b1 Meaning
0 0=MSD is not supported
0 RFU
1 1=qVSDC supported
X 1=EMV contact chip supported
X 1=Offline-only reader
X 1=Online PIN supported
X 1=Signature supported
X 1=Offline Data Authentication (ODA) for Online Authorizations supported
Byte 2
b8 b7 b6 b5 b4 b3 b2 b1 Meaning
0 1=Online cryptogram required
0 1=CVM required
X 1=(Contact Chip) Offline PIN supported
0 RFU
